windows-fzf

Description

A super fast fuzzy file finder for windows written in C#

How is this different to other fuzzy finders?

Instead of searching for files on disk (which is slow and very un-cool) it instead:

  1. Adds all file paths in working directoy to a SQLite db
  2. Loads sections of db into memeory
  3. Searces those section for search term fuzzily

which is far superior.

How it works

Creating the database

In order to search in a db we must first create the db using the command `fzf -createdb`, this creates an empty SQLite db in the `Documents\windows-fzf` folder.

An empty db is as good as no db

Now that the db has been created, we need to populate it. We do that by running `fzf -updatedb`. This starts three threads that begin drilling down into all directories and sub directories within the current working directory and adding every file path they encounter to the db.
